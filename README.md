# Compilador
Proyecto que implementa las bases de un compilador, incluyendo análisis de gramática, cálculo de conjuntos FIRST/FOLLOE y un parser LL() para el analis sintáctico.

## Objetivo

El objetivo del proyecto es comprender e implementar los conceptos del analisis lexico-sintactico en compiladores, enfocandose en:

- Representación formal de grámaticas
- Análisis de gramáticas libres de contexto
- Contrucción de parser LL(1)
- Analisis lexico mediante Tokens requeridos por el analisis lexico

El proyecto es de uso academico y experimentación, no al uso productivo.
